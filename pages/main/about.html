Created by Matthew Shirazi, B. Eng

This project was created using Docker, Kubernetes, Python, HTML, CSS, and JavaScript. The website runs entirely in a container on a Linux virtual machine.

For newer elections (2019 and 2021), voting data and shapefiles were retreived from Elections Canada.

If you notice a mistake, please send an email to support@canadianelections.net!

Pre-2019 federal district shape data retreived from:

Taylor, Zachary; Lucas, Jack; Kirby, J.P.; Hewitt, Christopher Macdonald, 2023, "Replication Data for “Canada’s Federal Electoral Districts, 1867–2021: New Digital Boundary Files and a Comparative Investigation of District Compactness.” Canadian Journal of Political Science.", https://doi.org/10.5683/SP3/4E8DCR, Borealis, V2; fedshapes_cbf_20221003.zip
